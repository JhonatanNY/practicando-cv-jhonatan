Mis Trabajos

Proyecto Simple de bus

Este proyecto es una aplicación backend construida con Spring Boot, donde se gestionan dos entidades principales: buses y marcas. Se desarrollaron endpoints para listar buses, agregar marcas y asignar marcas a los buses. La base de datos está organizada en dos tablas principales, y la API expone las funcionalidades necesarias para interactuar con los datos de estos objetos. Por parte del frontend, se utilizó React, implementando componentes para mostrar la lista de buses y marcas. La comunicación entre el frontend y el backend se realiza a través de llamadas a la API REST o servicio Rest.

Proyecto tienda de Guitarras (React)

Tienda de guitarras en React, con carrito de compras, manejo de estado, LocalStorage y despliegue en Netlify.

Proyecto Calculadora de consumo y propinas (React)

Aplicación desarrollada en React que permite calcular el costo total de un consumo incluyendo la propina. Se maneja el estado de la aplicación con useState, mostrando en tiempo real el total a pagar por persona. La interfaz es dinámica y fácil de usar, me ayudo a practicar conceptos de componentes, props y hooks en React. Desplegado en Netlify.

Proyecto Rastreador de Calorías (React)

Aplicación en React que permite registrar comidas y ejercicios, calcular calorías en esta app se maneja el estado con useReducer y se persiste los datos en LocalStorage. Desplegado en Netlify.
